Welcome to Nexus Agent

This installer will set up Nexus Agent on your Mac.

Nexus Agent is the on-device endpoint for AI traffic monitoring and policy enforcement. After installation it will:

  • Install a Network Extension that observes outbound AI traffic
  • Add a device CA to your System keychain (required for TLS inspection)
  • Run a background daemon that uploads compliance evidence to your team's Hub
  • Show a menu-bar status icon you can use to pause monitoring

You will be prompted to approve the system extension on first launch (System Settings → General → Login Items & Extensions → Network Extensions).
